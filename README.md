# Manger

A script for managing drives, files, resources, and more.

## Requirements

- Git
### Supported Distros
- Pop!
- Ubuntu
- Fedora
- Redhat
- Arch
- Opensuse
- CentOS
- Debian


### Additional Software

- `bpytop`
- `tilix`
- `ranger`
- `pv`

Ensure you have the necessary permissions to install these packages.

1. **Clone the repository:**
   ```sh
   git clone https://github.com/Timfedora/Manger.git
```sh
